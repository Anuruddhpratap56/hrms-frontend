# HRMS Lite Frontend (React)

## Setup
1. Install dependencies:
```powershell
npm install
```

2. Create `.env` from `.env.example`:
```env
VITE_API_BASE_URL=http://127.0.0.1:8000
```

3. Start app:
```powershell
npm run dev
```

## Features

### Employee Section
- Employee table with pagination
- Search by employee name
- Create employee (ID auto-generated by backend)
- Delete employee
- Row action buttons:
  - Mark/Unmark attendance for selected date (defaults to today if date not selected)
  - View employee attendance in popup
  - Delete
- Toasts for create and attendance actions/errors

### Attendance Section
- Attendance table with pagination
- Filters:
  - Date
  - Employee ID
  - Employee name search
- Row action button to mark/unmark for that employee-date
- Status badge for Present/Absent
- Toasts using employee name

## Reused UI Components
- `DataTable`
- `Modal`
- `PaginationControls`
- `StatusBadge`
- `ConfirmDialog`

## Notes
- Action buttons are icon-based with hover tooltip text
- Attendance page for selected date shows both present and absent employees
